# Як увімкнути українську мову на SCMDB

## Спосіб 1 — Через адресу (найшвидший)

Відкрийте в браузері:

```
https://scmdb.net/?lang=https://raw.githubusercontent.com/Vova-Bob/SCMDB_LANG/main/lang-global-4.7.0-live.11518367.json
```

Переклад увімкнеться автоматично і збережеться в браузері. Робити це достатньо один раз.

## Спосіб 2 — Через акаунт SCMDB

1. Зареєструйтесь або увійдіть на [scmdb.net](https://scmdb.net/)
2. Відкрийте **Settings** (Налаштування)
3. У поле для мовного файлу вставте:

```
https://raw.githubusercontent.com/Vova-Bob/SCMDB_LANG/main/lang-global-4.7.0-live.11518367.json
```

4. Збережіть — налаштування синхронізується на всіх пристроях.

---

## Як вимкнути переклад

- **Без акаунту**: очистіть дані браузера, або відкрийте `scmdb.net/?lang=clear`
- **З акаунтом**: видаліть URL у Settings

## Що перекладається

Місії, описи, локації, кораблі, предмети, фракції — усе ігрове контент. Інтерфейс сайту (кнопки, фільтри, заголовки) залишається англійською.

## Покриття

- **~95.8%** контенту перекладено
- **2 ключі** відсутні (працюють англійською)
- Неперекладені елементи автоматично показуються англійською
