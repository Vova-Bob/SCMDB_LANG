# 📋 Покрокова інструкція: GitHub Gist для SCMDB

## 🎯 Що потрібно зробити:

1. ✅ Відкрити GitHub Gist
2. ✅ Завантажити файл локалізації
3. ✅ Отримати Raw URL
4. ✅ Використати в SCMDB

---

## КРОК 1: Відкрийте GitHub Gist

### Варіант A: Якщо у вас є акаунт GitHub

1. Відкрийте в браузері:
   ```
   https://gist.github.com
   ```

2. Увійдіть, якщо потрібно

### Варіант B: Якщо немає акаунту (швидко створити)

1. Відкрийте: https://github.com/signup
2. Заповніть форму (1-2 хвилини):
   - Username (наприклад: vova-sc-ukrainian)
   - Email
   - Password
3. Підтвердьте email
4. Тепер відкрийте: https://gist.github.com

---

## КРОК 2: Завантажте файл

### Варіант A: Перетягування (найпростіше)

1. Відкрийте папку:
   ```
   F:\SCMDB_LANG\
   ```

2. Знайдіть файл:
   ```
   lang-global-4.7.0-ptu.11506930.json
   ```

3. Перетягніть його на сторінку Gist

### Варіант B: Через кнопку

1. Натисніть "Add file" → "Upload files"
2. Виберіть файл з папки `F:\SCMDB_LANG\`
3. Натисніть "Confirm upload"

---

## КРОК 3: Опублікуйте Gist

1. **Перевірте назву файлу** (вона може бути довга)
2. В полі "Description" можете написати:
   ```
   Ukrainian localization for SCMDB - Star Citizen Mission & Data Browser
   ```
3. **ВАЖЛИВО:** Натисніть кнопку **"Create public gist"**
4. Не "Create secret gist", а саме "Public"!

---

## КРОК 4: Отримайте Raw URL

Це найважливіший крок! 🔥

1. Після створення Gist, ви побачите файл
2. **Клікніть на ім'я файлу** (синє посилання)
3. Тепер ви бачите вміст файлу
4. У **правому верхньому куті** натисніть кнопку **"Raw"**
5. Браузер відкриє чистий JSON файл
6. **Скопіюйте URL з адресного рядка**

**URL виглядає приблизно так:**
```
https://gist.githubusercontent.com/vova-sc-ukrainian/abc123def456/raw/1234567890/lang-global-4.7.0-ptu.11506930.json
```

---

## КРОК 5: Використовуйте в SCMDB

### Відкрийте в браузері:

```
https://scmdb.dev?lang=ВАШ_RAW_URL
```

### Приклад:
```
https://scmdb.dev?lang=https://gist.githubusercontent.com/vova-sc-ukrainian/abc123def456/raw/1234567890/lang-global-4.7.0-ptu.11506930.json
```

### Кращий спосіб - через налаштування SCMDB:

1. Відкрийте: https://scmdb.dev
2. Натисніть на іконку ⚙️ (Settings) - зазвичай у правому верхньому куті
3. Знайдіть поле "Language" або "Localization"
4. Вставте ваш Raw URL
5. Збережіть налаштування

## 🔧 Налаштування збереження

URL локалізації **зберігається в браузері**, тому:
- Не потрібно додавати його щоразу
- Працює на всіх сторінках SCMDB
- Зберігається навіть після закриття браузера

---

## 🔄 Як оновити локалізацію в майбутньому

При новому патчі Star Citizen:

1. Запустіть:
   ```bash
   cd F:\SCMDB_LANG
   python auto_update_translation.py
   ```

2. Відкрийте ваш Gist на GitHub
3. Натисніть "Edit" на файлі
4. Видаліть старий вміст, вставте новий
5. Натисніть "Update gist"

**Raw URL залишається тим самим!** Нічого змінювати в SCMDB не потрібно!

## 🆘 Проблеми?

### Проблема: Не бачу кнопку "Raw"
**Рішення:** Клікніть на ім'я файлу спочатку, потім з'явиться кнопка Raw

### Проблема: SCMDB не показує українську
**Рішення:** Перевірте що URL починається з `https://gist.githubusercontent.com/` і закінчується на `.json`

### Проблема: Помилка CORS
**Рішення:** Переконайтеся що створили **Public gist**, а не Secret
